# Tressure-Hunt-Game
This project is a basic python game made primarily using if-else statements. I'm using it as a pathway for building relatively complex projects in the future.
COCEPTS USED 
- if-else statements
- user input
- conditional logic
- basic game flow
HOW TO RUN THIS GAME ON PYCHARM
1. Make sure Python 3 is installed on your computer.
2. Download or clone this repository.
3. Open PyCharm.
4. Click Open and select the project folder.
5. Open the file game.py (or whatever your main file is called).
6. Right-click inside the file and click Run 'game'
7. OR press the green Run button at the top.
8. The game will start in the Run terminal at the bottom of PyCharm.
HOW TO PLAY
1. Read the prompts carefully.
2. Type your choices when asked.
3. Try to find the treasure!
